### 용도
GitHub App을 계정에 설치한 뒤(로그인), 스타 증가를 원하는 퍼블릭 레포지토리를 선택해서 서버로 제출하면 서버에서 기존에 제출된 여러 계정의 토큰을 활용해 레포지토리들의 스타를 서로 증가시켜줌.

토큰은 GitHub App을 활용해 퍼블릭 레포지토리를 조회하고 스타를 누를수 있는 최소 권한만 가지고 있고, 만료 기한이 있어 자동적으로 만료됨.
